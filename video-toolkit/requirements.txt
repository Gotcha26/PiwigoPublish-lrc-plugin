# Video Toolkit — Dépendances Python
# Aucune dépendance externe requise.
# Tous les modules utilisés sont de la bibliothèque standard Python 3.8+
#
# Outils système requis (non-Python) :
#   - FFmpeg 5.0+ (ffmpeg + ffprobe)
#   - ExifTool 12+ (optionnel — copie de métadonnées)
#
# Installation outils :
#   Windows : winget install ffmpeg  OR  https://ffmpeg.org/download.html
#   macOS   : brew install ffmpeg exiftool
#   Linux   : apt install ffmpeg libimage-exiftool-perl
